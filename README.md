## ¿Qué es el Scope?

El "Scope" determina la accesibilidad de las variables, objetos y funciones desde diferentes partes del codigo.

### - Tipos de "Scope"

Global
Function
Block

### - Javascript Variables.

En javascript los objetos y las funciones también son variables.
